Remote Data Structures¶

Data structures are fundamental constructs around which you build your application. They are used in almost all computer programs, therefore becoming fluent in what the standard data-structures can do for you is essential to get full value out of them. The standard template library in C++ implements a wide array of structures, python has several of them too. stdnet implements four remote structures, all derived from stdnet.Structure.

Structure¶

class stdnet.Structure(cursor, id, timeout=0, pickler=None, pipeline=None, **kwargs)¶

Base class for remote data-structures.

cursor¶

instance of a stdnet.BackendDataServer.

id¶

unique id for the structure

timeout¶

Expiry timeout. If different from zero it represents the number of seconds after which the structure is deleted from the data server. Default 0.

pipeline¶

An instance of PipeLine.

size()¶

Number of elements in structure

delete()¶

Delete structure from remote server.

add_expiry()¶

Internal method called if a timeout is set. This needs to implemented.

List¶

class stdnet.List(cursor, id, timeout=0, pickler=None, pipeline=None, **kwargs)¶

A linked-list stdnet.Structure.

push_back(value)¶

Appends a copy of value to the end of the remote list.

push_front(value)¶

Appends a copy of value to the beginning of the remote list.

Set¶

class stdnet.Set(cursor, id, timeout=0, pickler=None, pipeline=None, **kwargs)¶

An unordered set stdnet.Structure. Equivalent to python set.

add(value)¶

Add value to the set

update(values)¶

Add iterable values to the set

discard(elem)¶

Remove an element from a set if it is a member

OrderedSet¶

class stdnet.OrderedSet(cursor, id, timeout=0, pickler=None, pipeline=None, **kwargs)¶

An ordered stdnet.Set.

add(value)¶

Add value to the set

HashTable¶

class stdnet.HashTable(*args, **kwargs)¶

A hash-table stdnet.Structure. Equivalent to a Python dict.

add(key, value)¶

Add key - value pair to hashtable.

update(mapping)¶

Add mapping dictionary to hashtable. Equivalent to python dictionary update method.

mget(keys)¶

Return a generator of key-value pairs for the keys requested

items()¶

Generator over key-value items

clear()¶

Clear the Hash table. Equivalent to dict.clear method in Python.
