Set¶

class stdnet.Set(cursor, id, timeout=0, pickler=None, pipeline=None, **kwargs)¶

An unordered set stdnet.Structure. Equivalent to python set.

This structure is used for in two different parts of the library.

  • It is the structure upon which indexes are built, therefore each stdnet.orm.Field which has index set to True will have an associated Set structure in the data server backend.
  • It is also used as stdnet.orm.SetField.

HashTable¶

class stdnet.HashTable(*args, **kwargs)¶

A hash-table stdnet.Structure. The networked equivalent to a Python dict.

This structure is important since it is used in two different parts of the library.

  • It is the structure which holds instances for a stdnet.orm.StdModel class. Therefore each model is represented as a HashTable structure. The keys are the model instances ids and the values are the serialised version of the instances.
  • It is also used as field (stdnet.orm.HashField) like all other class:stdnet.Structure.
