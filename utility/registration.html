Model Utilities¶

Instance form UUID¶

stdnet.orm.from_uuid(uuid)¶

Retrieve an instance of a stdnet.orm.StdModel from an universal unique identifier uuid. If the uuid does not match any instance an exception will raise.

Model Iterator¶

stdnet.orm.model_iterator(application)¶

Returns a generatotr of stdnet.orm.StdModel classes found in the models module of an application dotted path.

Parameters:application – An iterable over python dotted-paths where models are defined.

For example:

from stdnet.orm import model_iterator

APPS = ('stdnet.contrib.searchengine',
        'stdnet.contrib.timeseries')

for model in model_iterator(APPS):
    ...

Register Models¶

stdnet.orm.register_application_models(applications, models=None, app_defaults=None, default=None)¶

Returns a generator which register models in application.

Parameters:
  • application – A String or a list of strings which represent python dotted paths to modules containing a models module where models are implemented.
  • models – list of models to include or None (all models). Default None.

For example:

register_application_models('mylib.myapp')
stdnet.orm.register_applications(applications, **kwargs)¶

A simple convenience wrapper around the register_application_models() generator. It return s a list of registered models.

Flush Models¶

stdnet.orm.flush_models(includes=None, excludes=None)¶

Utility for flushing models data. It removes all keys associated with models.
