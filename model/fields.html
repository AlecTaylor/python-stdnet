Scalar Fields¶

The most important part of a model, and the only required part of a model, is the list of database fields it defines. Fields are specified by class attributes. They are the equivalent to django Fields and therefore the equivalent of columns in a traditional relational databases.

Field Base Class¶

class stdnet.orm.Field(unique=False, ordered=False, primary_key=False, required=True, index=True, default=<NoValue>, **extras)¶

This is the base class of all StdNet Fields. Each field is specified as a stdnet.orm.StdModel class attribute.

index¶

If True, the field will create indexes for fast search. An index is implemented as a stdnet.Set in the stdnet.BackendDataServer. If you don’t need to search the field you should set this value to False.

Default True.

unique¶

If True, the field must be unique throughout the model. In this case Field.index is also True. Enforced at stdnet.BackendDataServer level.

Default False.

ordered¶

If True, the field is ordered. if Field.unique is True this has no effect.

Default False.

primary_key¶

If True, this field is the primary key for the model. In this case Field.unique is also True.

Default False.

required¶

If False, the field is allowed to be null.

Default True.

default¶

Default value for this field.

Default novalue.

name¶

Field name, created by the orm at runtime.

model¶

The stdnet.orm.StdModel holding the field. Created by the orm at runtime.

to_python(value)¶

Converts the input value into the expected Python data type, raising stdnet.FieldValueError if the data can’t be converted. Returns the converted value. Subclasses should override this.

register_with_model(name, model)¶

Called during the creation of a the stdnet.orm.StdModel class when stdnet.orm.base.Metaclass is initialised. It fills Field.name and Field.model. This is an internal function users should never call.

serialize(value)¶

Called by the stdnet.orm.StdModel.save() method when saving an object to the remote data server. It return s a serializable representation of value. If an error occurs it raises stdnet.exceptions.FieldValueError

id(obj)¶

Field id for object obj, if applicable. Default is None.

has_default()¶

Returns a boolean of whether this field has a default value.

get_default()¶

Returns the default value for this field.

index_value()¶

A value which is used by indexes to generate keys.

AtomField¶

class stdnet.orm.AtomField(unique=False, ordered=False, primary_key=False, required=True, index=True, default=<NoValue>, **extras)¶

The base class for fields containing atoms. An atom is an irreducible value with a specific data type. it can be of four different types:

  • boolean
  • integer
  • date
  • datetime
  • floating point
  • symbol

IntegerField¶

class stdnet.orm.IntegerField(unique=False, ordered=False, primary_key=False, required=True, index=True, default=<NoValue>, **extras)¶

An integer AtomField.

to_python(value)¶

Convert value to an integer if possible.

BooleanField¶

class stdnet.orm.BooleanField(required=False, **kwargs)¶

An boolean AtomField

AutoField¶

class stdnet.orm.AutoField(unique=False, ordered=False, primary_key=False, required=True, index=True, default=<NoValue>, **extras)¶

An IntegerField that automatically increments. You usually won’t need to use this directly; a primary key field will automatically be added to your model if you don’t specify otherwise.

FloatField¶

class stdnet.orm.FloatField(*args, **kwargs)¶

An floating point AtomField. By default its Field.index is set to False.

DateField¶

class stdnet.orm.DateField(unique=False, ordered=False, primary_key=False, required=True, index=True, default=<NoValue>, **extras)¶

An date AtomField represented in Python by a datetime.date instance.

SymbolField¶

class stdnet.orm.SymbolField(unique=False, ordered=False, primary_key=False, required=True, index=True, default=<NoValue>, **extras)¶

An AtomField which contains a symbol. A symbol holds a sequence of characters as a single unit. A symbol is irreducible, and are often used to hold names, codes or other entities. They are indexes by default.

CharField¶

class stdnet.orm.CharField(*args, **kwargs)¶

A text Field which is never an index. It contains strings and by default Field.required is set to False.

ForeignKey¶

class stdnet.orm.ForeignKey(model, related_name=None, **kwargs)¶

A field defining a one-to-many objects relationship. Requires a positional argument: the class to which the model is related. To create a recursive relationship, an object that has a many-to-one relationship with itself, use:

orm.ForeignKey('self')

It accepts related_name as extra argument. It is the name to use for the relation from the related object back to self. For example:

class Folder(orm.StdModel):
    name = orm.SymobolField()

class File(orm.StdModel):
    folder = orm.ForeignKey(Folder, related_name = 'files')
