Timeseries Models¶

This is an experimental module which requires redis-timeseries branch of redis. For more information check the redis timeseries API. To run the timeseries tests:

python runtests.py timeseries -i ts

number of dates in timeseries

intervals(startdate, enddate, parseinterval=None)¶

Given a startdate and an enddate dates, evaluate the date intervals from which data is not available. It return a list of two-dimensional tuples containing start and end date for the interval. The list could countain 0,1 or 2 tuples.
