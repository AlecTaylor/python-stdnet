A Search Engine for Stdnet Models¶

An application for building a search-engine on stdnet models with ideas from the fast, fuzzy, full-text index with Redis blog post and @antirez autocomplete https://gist.github.com/574044.

Usage¶

Somewhere in your application create the search engine singletone:

from stdnet.contrib.searchengine import SearchEngine
 
engine = SearchEngine()

To register a model with the search engine:

engine.register(MyModel)

API¶

class stdnet.contrib.searchengine.SearchEngine(min_word_length=3, stop_words=None, autocomplete='en', metaphone=True)¶

Search engine driver. Adapted from https://gist.github.com/389875

Parameters:
  • min_word_length –

    minimum number of words required by the engine to work.

    Default 3.

  • stop_words –

    list of words not included in the search engine.

    Default in stdnet.contrib.searchengine.ignore.STOP_WORDS

  • autocomplete –

    Name for the autocomplete sorted set. If None autocomplete functionality won’t be available.

    Default en.

  • metaphone –

    If True the double metaphone algorithm will be used to store and search for words.

    Default True.

To use it:

from stdnet.contrib.searchengine import SearchEngine
 
engine = SearchEngine()
register(model, order_by=None)¶

Register a model to the search engine. By registering a model, every time an instance is updated or created, it will be indexed by the search engine.

Parameters:
  • model – a stdnet.orm.StdModel class.
  • order_by – an optional list of fields used to order the results once available.
index_item(item)¶

Extract content from the given item and add it to the index. If autocomplete is enabled, it adds indexes for it.

Parameters:item – an instance of a stdnet.orm.StdModel.
remove_item(item)¶

Remove indexes for item.

Parameters:item – an instance of a stdnet.orm.StdModel.
search(text, **filters)¶

Full text search

add_tag(item, text)¶

A a tag to an object. If the object already has the tag associated with it do nothing.

Parameters:
  • item – instance of stdnet.orm.StdModel.
  • tag – a string for the tag name or a Tag instance.

It returns an instance of TaggedItem.

alltags(*models)¶

Return a dictionary where keys are tag names and values are integers representing how many times the corresponding tag has been used against the Model classes in question.

get_words_from_text(text)¶

A generator of words to index from the given text

get_metaphones(words)¶

Get the metaphones for a given list of words
