Python Stdnet¶

A networked data structures manager design to work with Redis data-store, but implemented so that other storage systems can be supported in the future.

It is shipped with an object relational mapper which greatly facilitates the management and retrieval of large data-sets. It includes a stand-alone, python 3 compatible, redis client which was originally forked from redis-py. There are no dependencies, it requires python 2.6 up to python 3.3 and there are over 500 tests with a coverage of about 90%.

The library is stable, used in production and continuously maintained.

Dependencies: None

Optional Dependencies: cython for the Redis C parser.
